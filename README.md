## 🚀 Öne Çıkan Projeler

### 🔍 RAG Mimarisi ile Veri Analizi ve Soru-Cevap Platformu
**Eylül 2025 - Ekim 2025**

AI-powered bir veri analiz platformu geliştirdim. Herhangi bir CSV veri seti üzerinde doğal dil ile sorgulama yapılabiliyor.

**🎯 Temel Özellikler:**
- ✨ RAG (Retrieval-Augmented Generation) mimarisi ile semantic search
- 📊 Otomatik EDA, istatistiksel özetler ve korelasyon analizi
- 📈 10+ interaktif görselleştirme seçeneği (histogram, heatmap, scatter matrix, vb.)
- 🤖 GPT-3.5-turbo entegrasyonu ile AI-destekli grafik yorumlama
- 💡 Context-aware yanıtlama sistemi

**🛠️ Teknolojiler:** Python, Streamlit, LangChain, Sentence Transformers, Pandas, Plotly, ChromaDB

---

### 🎯 Jira Agent - AI Destekli Görev Analiz Sistemi
**Eylül 2025 - Ekim 2025**

Jira görev yönetimini otomatikleştiren AI destekli bir agent sistemi.

**🎯 Temel Özellikler:**
- 🔍 Task açıklamalarını ve yorumları otomatik analiz
- ⚠️ Risk tespiti ve aksiyon önerileri
- 📋 Günlük/Haftalık/Öncelik analizi raporları
- 📧 Otomatik email raporlama sistemi
- 🔗 Jira REST API ve Gmail SMTP entegrasyonu

**🛠️ Teknolojiler:** Python, LangChain, OpenAI GPT-4o, Jira REST API

---

### 💼 Distribütör İş Takip Portalı - Kurumsal ERP Sistemi
**Ekim 2024 - Mart 2025 | Bitirme Projesi**

Distribütör firmaların tüm operasyonel süreçlerini yöneten kapsamlı bir ERP portalı.

**🎯 Temel Özellikler:**
- 🏢 6 farklı operasyonel modül (Stok, Satış, Nakliye, Faturalama, Sigorta, Gümrük)
- 🔐 ASP.NET Identity ile rol bazlı yetkilendirme
- 👥 Admin paneli ile kullanıcı yönetimi
- 🗄️ Katmanlı mimari ve Entity Framework Core
- 📱 Responsive Razor Pages frontend

**🛠️ Teknolojiler:** ASP.NET Core MVC, Entity Framework Core, SQL Server, Razor Pages

---

### 🎓 Çalışanların İşten Ayrılma Eğilimi Tahmini
**Ekim 2024 - Ocak 2025**

Machine learning ile çalışan turnover tahmin sistemi.

**🎯 Temel Özellikler:**
- 📊 55.274 kayıt ve 26 değişken içeren veri seti analizi
- 🤖 7 farklı ML modeli karşılaştırmalı analizi (Random Forest, SVM, XGBoost, LSTM, GRU vb.)
- 🎯 **%95.72 doğruluk** ve **%95.32 F1 skoru** ile Stacked Ensemble modeli
- 💼 İK yöneticileri için karar destek sistemi
- 📈 Feature importance analizi ve yorumlanabilir sonuçlar

**🛠️ Teknolojiler:** Python, TensorFlow, Keras, Scikit-learn, Pandas, NumPy
